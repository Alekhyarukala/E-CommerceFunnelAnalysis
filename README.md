# 📊 E-commerce Funnel & Conversion Dashboard

## 🔍 Project Overview

This project analyzes user behavior across an e-commerce funnel to understand how users move from **views → cart → purchase**.

The goal is to identify:

* Drop-off points in the funnel
* Conversion performance
* Revenue insights
* User behavior patterns

---

## 🎯 Objectives

* Analyze **traffic-to-conversion journey**
* Measure **conversion rates at each stage**
* Identify **cart abandonment issues**
* Provide **actionable business insights**

---

## 🛠️ Tools & Technologies

* **Python**
* **Streamlit** (Dashboard)
* **Pandas** (Data Processing)
* **Plotly** (Visualization)

---

## 📊 Dashboard Features

* 📌 KPI Metrics (Users, Views, Conversion Rate, Revenue)
* 🔻 Funnel Analysis (User drop-off)
* 📈 Revenue Trend Over Time
* ⏰ Hourly User Activity Analysis
* 🛍️ Category-wise Conversion Performance
* 💰 Price Distribution Insights

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate to the project folder:

```bash
cd your-project
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the dashboard:

```bash
streamlit run app.py
```
